Landing page moderna para una agencia digital enfocada en impulsar el crecimiento de pequeñas y medianas empresas a través de soluciones tecnológicas estratégicas.

## Tecnologías utilizadas

- React
- TypeScript
- TailwindCSS
- Framer Motion
- React Parallax Tilt

##  Características

- Diseño moderno y responsive
- Animaciones suaves y profesionales
- Hero section con efecto 3D interactivo
- Arquitectura de componentes reutilizables
- Código organizado y escalable

## Objetivo del proyecto

Construir una interfaz atractiva y estratégica que comunique profesionalismo, innovación y enfoque en resultados para empresas en crecimiento.
